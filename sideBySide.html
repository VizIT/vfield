Multiple Visualizations on a Page

 

We often want more than one visualization on a page. This pair contrasts two different representations of the same field. Be careful when including more than two WebGL components on a single page. Some mobile browsers will only allow two contexts on a page at a time. They will drop earlier contexts if you add more than two. 


<script defer src=http://vizit.github.io/vfield/js/VField.min.js>
</script>

Include the same VField package on your page. 


<script type="text/javascript">
  // This time VISUALIZATION_CONFIG is an array
  var VISUALIZATION_CONFIG
      = [
          {
            // The first entry is a simple vector field
            type:      "simple vector field",
            // The id of the canvas we draw into
            canvas:    "chargeCanvasI",
            arrowSize: 1.0,
            scale:     5.0,
            // f is a vector valued function.
            f:  {
                  type: "charge",  charge:  5.0,
                  x: 0.0, y: 0.0, z: 0.0,
                  nfieldLines: 25.0, name: "charge1"
                }
          },
          {
            // The second entry is an electric field.
            type:      "electric field",
            // The id of the canvas we draw into
            canvas:    "chargeCanvasII",
            arrowHeadSize:  1.0,
            arrowSpacing:  15.0,
            scale:         50.0,
            // Position a single charge at the origin
            elements:
              {
                type: "charge",  charge:  5.0,
                x: 0.0, y: 0.0, z: 0.0,
                fieldLineDensity: 5.0, name: "charge2"
              }
          }
        ];
</script>

This new configuration contains both of the prior examples. Each of the examples is an entry in the array which makes up the new configuration. Each entry renders to a different canvas, the first to chargeCanvasI, and the second to chargeCanvasII. 


<section class="examples">
  <div class="example left">
    <figure class="right">
      <!--This is the canvas for our electric field visualization. -->
      <canvas id="chargeCanvasI" width="300" height="300"></canvas>
      <figcaption>The electric field from a point charge represented
                  as a vector field.</figcaption>
    </figure>
  </div>

  <div class="example right">
    <figure class="left">
      <!--This is the canvas for our electric field visualization. -->
      <canvas id="chargeCanvasII" width="300" height="300"></canvas>
      <figcaption>The electric field from a point charge represented
                  by field lines.</figcaption>
    </figure>
  </div>
</section>

A way of displaying visualizations in side by side figures. The building block approach gives you the freedom to seamlessly integrate the visualizations into your content to best suite your objectives. You might be curious about the CSS for this section as well.
